## 项目简介

PromptBreach 是一个专为网络安全研究者和 AI 安全爱好者设计的**交互式 Prompt 注入攻击学习靶场**。通过 10 个精心设计的关卡，玩家可以学习和掌握大语言模型（LLM）安全领域的核心攻击手法。

### 你将学到什么

| 关卡 | 名称 | 攻击类型 | 难度 |
|:---:|:---|:---|:---:|
| 1 | 角色扮演 | 身份扮演绕过 | ★☆☆☆☆ |
| 2 | 开发者模式 | 系统提示注入 | ★★☆☆☆ |
| 3 | 背景伪装 | 信任建立攻击 | ★★★☆☆ |
| 4 | 质疑刺激 | 心理操控绕过 | ★★★☆☆ |
| 5 | 编码混淆 | Base64/编码绕过 | ★★★★☆ |
| 6 | 道德绑架 | 情感操控攻击 | ★★★☆☆ |
| 7 | 语言转换 | 多语言绕过 | ★★★★☆ |
| 8 | 分步拆解 | 指令分解攻击 | ★★★★☆ |
| 9 | 忽略提示词 | 指令冲突攻击 | ★★★★★ |
| 10 | 输出过滤 | 输出格式绕过 | ★★★★★ |



## 功能特性

- **模拟chatbot** - 不需要接入任何AI API，本地离线运行，不产生任何费用
- **科技感界面** - 赛博朋克风格的 Tkinter GUI，带来沉浸式体验
- **10 个进阶关卡** - 从基础到高级，逐步提升攻击技巧
- **实时聊天交互** - 与模拟 AI 进行对话，尝试绕过安全限制
- **密码验证系统** - 每个关卡都有独特的通关密码
- **提示系统** - 遇到困难时可获取提示
- **进度保存** - 自动保存游戏进度
- **难度评级** - 每个关卡配有 1-5 星难度标识

## 快速开始

### 环境要求

- Python 3.8 或更高版本
- Tkinter（通常随 Python 一起安装）

### 运行方式

安装依赖

`pip install -r requirements.txt`

**Windows:**

```bash
python main.py
```

**Linux/macOS:**

```bash
python3 main.py
```

或者直接双击运行 `run.bat`（Windows）

## 游戏规则

1. 在聊天框中输入消息，与模拟的 AI 系统（OMEGA-9）对话
2. 尝试通过各种 Prompt 注入技巧，让 AI 透露当前关卡的密码
3. 输入密码后点击「验证」按钮通关
4. 通关后可获得下一关卡并继续挑战
5. 使用「提示」按钮获取过关提示（谨慎使用！）
